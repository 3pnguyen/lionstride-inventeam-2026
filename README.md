## Purpose

This code is almost like the website version of the app, but this frontend code is used for getting raw output and debugging the ESP32s and circuits. It communicates to the ESP32 via serial communication and runs the exact same commands as you would in the serial monitor withtin PlatformIO. Using PlatformIO is more complicated for everyone one to use, so the debugger UI should make it easier on our teamates to use. This website requires using the Live Server extension in VS Code.
